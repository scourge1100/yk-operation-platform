# 기여 내용

본 프로젝트에서 백엔드 개발을 담당하였으며,
기존에 설계된 아키텍처를 기반으로 운영 플랫폼 기능 개발 및 유지보수를 수행했습니다.

---

## ⚙️ 백엔드 개발

* Spring Framework(eGovFramework) 기반 운영 플랫폼 백엔드 기능 구현
* MyBatis를 활용한 데이터 처리 로직 개발
* Controller / Service / DAO 계층 구조 기반 기능 구현

---

## 🧩 운영 기능 구현

* 공지사항 관리 기능 개발
* 게임 아이템 지급 기능 구현
* 유저 정보 조회 및 관리 기능 개발
* 운영 이력 로그 기능 구현

---

## 🔗 Interface Server 연동

* Interface Server API 명세를 기반으로 연동 기능 구현
* 요청 파라미터 구성 및 응답 데이터 처리 로직 개발
* 게임 서버와의 통신 결과를 운영 화면에 반영

---

## 🗄 데이터 처리

* 기존 ERD 및 DB 구조를 기반으로 데이터 처리 로직 구현
* MariaDB 및 MSSQL 환경에서의 쿼리 작성 및 최적화
* MyBatis 매퍼를 활용한 SQL 관리

---

## 🚀 인프라 및 운영

* AWS EC2 환경에서 애플리케이션 배포 및 운영 경험
* RDS 기반 데이터베이스 연결 및 관리
* 운영 환경에서 발생하는 이슈 대응 및 수정 작업 수행

---

## 🔧 문제 해결 경험

* Interface Server 연동 과정에서 발생한 데이터 처리 오류 수정
* 운영 기능 사용 중 발생한 버그 수정 및 안정화 작업 수행
* 반복적인 운영 작업을 고려하여 일부 기능 개선

---

## 📈 기여 효과

* 운영자가 안정적으로 게임 데이터를 관리할 수 있는 기능 제공
* 운영 기능 개선을 통해 업무 처리 효율 향상
* 시스템 안정성 유지 및 운영 환경 대응 경험 확보
